La utilización del Sitio Web atribuye la condición de usuario del Sitio Web (en adelante, el "Usuario") y expresa la aceptación plena y sin reservas del Usuario de todas y cada una de las Condiciones Generales en la versión publicada por ASEJA.

Con anterioridad a la utilización de dichos servicios, por tanto, el Usuario también ha de leer atentamente las correspondientes Condiciones Particulares.
